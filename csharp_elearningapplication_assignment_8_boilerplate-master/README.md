# Junior FSE Pre-Req PRMC1 - Elearning Application Assignment

## ELearningApplication_Assignment_8 `NUnit Testing`

## Problem Statement

`ELearningApplication` is a Console Application that allows users to subscribe for Online Courses

The application handles Course management operations and Subscriber management operations

The application is required to be designed based on TDD approach.

The TDD approach involves writing test cases first based on requirements provided and then build the solution code in conformance with the test cases.



***Tech Stack***
1. C#
2. .NET Framework 4.8
3. NUnit Testing
